## No Front foi desenvolvida uma aplicação SPA utilizando a biblioteca `ReactJS` e a linguagem `Typescript`

### Requisitos:

- NodeJS > 8
- npm > 5 ou yarn

### Para rodar o projeto uttilize os eguintes comandos:

```sh
cd frontend

yarn install
# ou
npm install

yarn start
# ou
npm start
```

## acesse http://localhost:3000
